Vending Machine Logic (C++)
Цей проєкт моделює базову логіку роботи торговельного автомата та перевірку обмежень за двома параметрами: бюджет користувача та наявність товару в автоматі.

Технології:
C++.

Бібліотека <algorithm> — використана для ефективного знаходження мінімального значення між двома показниками.

Бібліотека <windows.h> — для коректної підтримки кириличного тексту в консолі.

Що робить ця програма:
Аналізує купівельну спроможність: Програма обчислює, скільки одиниць товару користувач може дозволити собі за наявний бюджет.

Враховує обмеження залишків: Алгоритм порівнює максимально можливу кількість покупок із реальною кількістю пляшок, що залишилися в автоматі.

Видає точний результат: Використовує функцію min(), щоб визначити фінальну кількість пляшок, яку клієнт дійсно зможе отримати.
